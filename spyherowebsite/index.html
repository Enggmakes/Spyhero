Workflow

From Raw Text to Perfect Prompt
in One Keystroke

SpyHero operates as an invisible layer between you and your AI tool. Here's exactly what happens in under two seconds.

1

Select Your Text

Highlight any rough draft, quick note, or prompt in any application — your email client, browser, code editor, chat app, or document editor. SpyHero works with everything.

2

Press Your Hotkey

Hit your configured global shortcut (default: Ctrl + Shift + P) from anywhere, without switching windows or losing focus on your work.

Registered at OS level via Electron globalShortcut
3

Intelligent Text Capture

SpyHero silently sends Ctrl+C to your active window using a native VBScript — capturing your selected text without disrupting your cursor position or application focus. If no text is selected, a clean manual input form opens instead.

VBScript keystroke simulation — non-disruptive
4

Choose Enhancement Mode

A sleek floating overlay appears. Your text is pre-loaded. Select from 8 expert enhancement modes — General, Developer, Marketing, Creative, and more — each powered by a precisely engineered system prompt.

5

AI Processes Your Prompt

Your text is sent directly from SpyHero to your chosen AI provider (Gemini, OpenAI, Claude, Groq, or OpenRouter) via a secure API call. No intermediate servers. No logging. Just your keys, your data, your provider.

Direct API call — zero data intermediaries
6

Enhanced Prompt Returned

The AI-optimized result appears in the overlay instantly. It's automatically copied to your clipboard. You can review it, re-run with a different mode, or edit the original.

7

Auto-Pasted Back — Seamlessly

With Auto-Paste enabled, SpyHero automatically injects the enhanced text back into your original application using Ctrl+V via VBScript. The overlay closes. You continue working — with a far better prompt already in place.

Fully configurable — toggle Auto-Paste in Settings
Features

Everything You Need.
Nothing You Don't.

SpyHero is purpose-built for speed, privacy, and flexibility. Every feature is designed around a real workflow benefit.

Global Hotkey Trigger

Activate SpyHero from any application using a fully customizable global keyboard shortcut — registered at the OS level so it works regardless of what window is active. Default: Ctrl+Shift+P. Record any combo you prefer in Settings.

Intelligent Text Capture

SpyHero captures your selected text silently using a native VBScript Ctrl+C keystroke — no clipboard conflict, no focus loss. If nothing is selected, a manual input form opens automatically. Pre-fills from clipboard if relevant text is present.

Auto Copy & Paste Loop

The enhanced result is automatically copied to clipboard the moment enhancement completes. With Auto-Paste enabled, it's injected back into your original app via VBScript — creating a seamless, near-instant enhancement loop with zero manual steps.

System Tray Integration

SpyHero runs completely silently in the background — no taskbar entry, no windows in the way. Right-click the tray icon to access settings, trigger enhancement, or quit. The tray menu dynamically displays your active shortcut, updating whenever you change it.

Comprehensive Settings Panel

Full control in one place: record custom shortcuts, switch AI providers, enter API keys (stored locally), select or type model names, adjust the creativity temperature slider (0.0–1.2), toggle auto-paste, enable launch on startup, and clear prompt history.

Prompt History

Every enhancement is saved locally (up to 20 entries) in browser storage. Access the History panel from the overlay header to browse, reload, or reuse any past original + enhanced prompt pair, along with the mode it was generated in.

Why BYOK?

Bring Your Own Key means you connect SpyHero directly to your chosen AI provider. There's no SpyHero middleman, no markup on API calls, and no third-party ever sees your data. You pay your provider's standard rate — often fractions of a cent per enhancement.

No Backend Server

SpyHero has no server, no cloud infrastructure, and no telemetry. All AI calls are made directly from the Electron renderer via fetch() to your provider's API.

Local Settings Storage

All configuration — API keys, provider choice, shortcut, preferences — is stored in a plain JSON file at %AppData%/SpyHero/settings.json on your machine only.

Prompt Injection Protection

User text is wrapped in <prompt_to_enhance> XML tags with explicit instructions — preventing the AI from accidentally executing your prompt instead of refining it.

Data Leaves Only to Your Provider

The only network traffic SpyHero generates is the direct API call to your chosen AI provider. Your prompts are not logged, analyzed, or retained by SpyHero in any way.

Instant Escape

Press Esc at any time to instantly close the overlay and return to your original application — no clicks, no delays.
